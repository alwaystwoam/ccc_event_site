// セクション 01 — WHY

Why this exists

The old world

Learn syntax first, ship later. Gatekeeping by tooling complexity. Months of bootcamp before you make anything real.

The new world

Describe intent clearly, ship now. AI translates your thinking into working code—today.

AI is not just "autocomplete." It's a general-purpose thinking partner that can:

  • Translate structured English into working code
  • Critique your UX like a harsh reviewer
  • Brainstorm features, edge cases, and product strategy
  • Help you debug when things break

This camp exists because the ability to build is no longer reserved for "traditional engineers." If you can think clearly, you can build tools that solve your real problems—and maybe discover a new career path along the way.

AI amplifies you.
If you're clear, it multiplies clarity.
If you're vague, it multiplies chaos.

    • Agent mode — execute. The AI writes, edits, and runs code across your project. This is where things get built

What to bring + setup

Bring

  • MacBook (required)
  • Charger
  • Reliable internet access (or hotspot)
  • 20 GB free disk space minimum
  • Optional: headphones

Before you arrive

Operating System

  • macOS updated to latest stable

  • GitHub account (logged in)
  • Vercel account (sign in with GitHub)
  • Anthropic account (for Claude access)
  • Cursor account (logged in)

Installed

  • Git installed
git --version
  • Node.js (LTS) installed
node -v && npm -v
  • Cursor installed + logged in

  • You can create a GitHub repo
  • You can clone it locally
  • You can run a simple Node command
